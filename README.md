# Polish Grammar Exercises / Упражнения по польской грамматике / Ćwiczenia z gramatyki polskiej

---

## 🇬🇧 English

Interactive grammar exercises based on the textbook **Bartłomiej Maliszewski, "Gramatyka z kulturą. Przez osoby"** (Lublin, 2023).

### What's inside

The exercises cover all 12 chapters of the textbook, from the present tense through past, future, conditional, imperative, passive voice, and verb government:

| Chapter | Topic |
|---------|-------|
| I | Present tense — *Już jest czas teraźniejszy!* |
| II | Aspectual pairs — *Tego trzeba się (na)uczyć* |
| III | Past tense — *Tu byłem!* |
| IV | Future tense — *Tu będzie czas przyszły* |
| V | Reflexive forms — *Formy z "się"?* |
| VI | Imperative mood — *Poćwiczmy tryb rozkazujący!* |
| VII | Conditional mood — *Może poćwiczylibyśmy tryb warunkowy?* |
| VIII | Reported speech — *Mowa zależna* |
| IX | Participles — *Często używane imiesłowy* |
| X | Passive voice — *Strona bierna* |
| XI | Impersonal forms — *Formy nieosobowe* |
| XII | Verbs and their collocations — *Czasowniki i ich sąsiedzi* |

### Exercise types

- **Fill-in-the-blank** — type the correct grammatical form
- **Matching** — connect items from two columns
- **Multiple choice** — select the correct option
- **Table completion** — fill in conjugation/declension paradigms
- **Odd-one-out** — find the word that doesn't fit
- **Word arrangement** — put words in the correct order

### How to use

Open `exercises/index.html` in a browser — no server or installation required. All exercises run locally.

## 🇷🇺 Русский

Интерактивные упражнения по польской грамматике на основе учебника **Бартломей Малишевский, «Gramatyka z kulturą. Przez osoby»** (Люблин, 2023).

### Содержание

Упражнения охватывают все 12 глав учебника: настоящее, прошедшее и будущее время, виды глаголов, повелительное и условное наклонение, страдательный залог, возвратные формы и управление глаголов.

### Типы упражнений

- **Вставить слово** — ввести правильную грамматическую форму
- **Соединить колонки** — связать элементы левой и правой колонок
- **Выбрать вариант** — выбрать правильный из предложенных
- **Заполнить таблицу** — вписать формы в парадигму
- **Найти лишнее** — определить слово, не вписывающееся в ряд
- **Расставить слова** — составить предложение из перемешанных слов

### Как использовать

Открой файл `exercises/index.html` в браузере. Сервер и установка не нужны — всё работает локально.

## 🇵🇱 Polski

Interaktywne ćwiczenia z gramatyki języka polskiego oparte na podręczniku **Bartłomiej Maliszewski, „Gramatyka z kulturą. Przez osoby"** (Lublin, 2023).

### Zawartość

Ćwiczenia obejmują wszystkie 12 rozdziałów podręcznika: czas teraźniejszy, pary aspektowe, czas przeszły i przyszły, formy z „się", tryb rozkazujący i warunkowy, mowę zależną, imiesłowy, stronę bierną, formy nieosobowe oraz łączliwość czasowników.

### Typy ćwiczeń

- **Uzupełnij lukę** — wpisz poprawną formę gramatyczną
- **Połącz kolumny** — dopasuj elementy z lewej i prawej kolumny
- **Zaznacz odpowiedź** — wybierz poprawny wariant
- **Uzupełnij tabelkę** — uzupełnij paradygmat odmiany
- **Które słowo nie pasuje?** — wskaż wyraz, który nie należy do grupy
- **Ułóż zdanie** — ułóż słowa we właściwej kolejności

### Jak używać

Otwórz plik `exercises/index.html` w przeglądarce. Nie jest potrzebny serwer ani instalacja — wszystko działa lokalnie.
